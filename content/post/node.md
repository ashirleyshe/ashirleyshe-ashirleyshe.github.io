## Install nvm
<https://github.com/nvm-sh/nvm>
```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
```

check nvm version
```
nvm --version
```

## Install node
```
nvm install node
```
check node version
```
node -v
```
other commands
```
nvm ls-remote
nvm ls
nvm install v10.6.3
nvm use v10.6.3
```
